# Xamarin.Forms samples

A series of samples that introduce **Xamarin.Forms** to Windows Phone / XAML developers,
mapping familiar concepts to the cross-platform Xamarin.Forms model.

> **Historical sample.** Xamarin.Forms has been succeeded by **.NET MAUI**. These
> projects are kept for reference; APIs and tooling have since changed.

## What's included

| Folder | What it demonstrates |
| --- | --- |
| `Dependency-MVVMLight` | Platform-specific services via the `DependencyService`, combined with MVVM Light for view-model wiring. |
| Additional folders | Further Xamarin.Forms topics (layouts, data binding, navigation). |

Each sample contains the shared Xamarin.Forms project plus Android and Windows Phone heads.

## Prerequisites

- Visual Studio 2015 / 2017 with the **Mobile development with .NET (Xamarin)** workload
- Android SDK / emulator

## Getting started

1. Clone the repository.
2. Open the `.sln` inside the sample folder you want to run.
3. Restore NuGet packages, choose a platform head and run.
